Importação do Bootstrap via CDN

O uso de um **CDN (Content Delivery Network)** para importar o Bootstrap oferece várias vantagens, como a redução do tempo de carregamento, pois os arquivos são entregues a partir de servidores distribuídos globalmente, mais próximos do usuário. Além disso, como muitas páginas utilizam o mesmo CDN para bibliotecas populares, os navegadores podem armazenar em cache esses arquivos, economizando largura de banda e acelerando o carregamento em visitas subsequentes.

No entanto, existem também desvantagens. Caso o CDN esteja fora do ar ou o usuário tenha problemas de rede, o carregamento da página pode ser afetado. Além disso, se você estiver dependendo de uma versão específica do Bootstrap, pode não ter controle total sobre as atualizações do arquivo no CDN. Por fim, se a conexão do usuário com o CDN for lenta, isso pode afetar negativamente a performance da página.
